The OpenGL Objects Wrapper Library (globjects) provides an object oriented C++ interface for OpenGL's programmable pipeline (3.0+). globjects is a cross-platform library licenced under the MIT license. Download here

Dependencies

The following dev-libraries and programs need to be provided for correct CMake configuration:
  • C++11 compatible compiler (e.g. gcc >= 4.7, clang >= 3.4, MSVC >= 2013 Update 3)
  • CMake (>= 2.8.12): http://www.cmake.org/
  • OpenGL C++ Bindings (glbinding >= 1.0.0): https://github.com/hpicgs/glbinding
  • OpenGL Mathematics (GLM >= 0.9.5): http://glm.g-truc.net/
  • Window and Context creation (GLFW >= 3.0.4): http://www.glfw.org/ (optional, needed for examples)
